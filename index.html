Webhook de envio

Quando o cliente enviar o formulário, as respostas serão enviadas para este endereço.
Como conectar ao Notion gratuitamente:
Crie uma conta em make.com (plano gratuito)
Crie um novo cenário → escolha Webhooks → Custom webhook como trigger
Copie a URL gerada e cole no campo acima
Adicione um módulo Notion → Create a database item e conecte à sua base
Mapeie os campos: client_name, responses[n].question, responses[n].answer
Ative o cenário e salve aqui
